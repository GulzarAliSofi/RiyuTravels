## Quick start

1. Install dependencies:

   npm install

2. Run dev server:

   npm run dev

3. Open http://localhost:5173

## Notes

- Images are now linked to curated Unsplash photos using the Unsplash Source API (featured queries).
- To receive form submissions, sign up at https://formspree.io and replace the `FORMSPREE_ENDPOINT` value in `src/pages/Contact.jsx` and `src/pages/PackageDetail.jsx` with your form endpoint (format: `https://formspree.io/f/your-id`).
- This is a starter scaffold; adapt styles, copy, and packages as needed.
